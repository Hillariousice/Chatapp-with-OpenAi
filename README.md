## Overview

This project explores integrating large language model capabilities into a real-time chat interface, combining a custom client and server architecture with OpenAI's API for dynamic, AI-powered conversations.

## Features

- 🤖 AI-powered responses via OpenAI API integration
- 💬 Real-time chat interface
- 🎨 Styled with SCSS for a clean, custom UI
- 🏗️ Separated client/server architecture

## Tech Stack

- **Frontend:** React, JavaScript
- **Styling:** SCSS
- **AI Integration:** OpenAI API
- **Backend:** Node.js (server folder)

## Getting Started

**Server:**
```bash
cd server
npm install
npm start
```

**Client:**
```bash
cd client
npm install
npm start
```

You'll need an OpenAI API key set in your environment variables to run this locally.
